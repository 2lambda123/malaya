# Malaya
NLTK like for bahasa Malaysia

## Requirements
  * NumPy >= 1.0
  * Sklearn >= 0.18
  * Fuzzywuzzy
  * unidecode
  * nltk
  * scipy

## Installation

```bash
pip3 install sklearn fuzzywuzzy nltk unidecode numpy scipy
```
